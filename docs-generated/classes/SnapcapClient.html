    Class SnapcapClient

    @snapcap/native — browser-free Snap client.

    Loads Snap's web JavaScript bundle and WASM modules directly in Node, with shimmed Chrome APIs so the bundle "thinks" it's still running in Chromium. Runs many accounts on a fraction of the resources Playwright would require.
