# Function: setLogger()

```ts
function setLogger(fn: Logger | undefined): void;
```

Defined in: [logging.ts:132](https://github.com/CodingButter/snapcap-native/blob/main/src/logging.ts#L132)

Install (or clear) the active logger.

## Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `fn` | [`Logger`](../type-aliases/Logger.mdx) \| `undefined` | The handler to install, or `undefined` to disable logging entirely (in which case the internal emit point becomes a one-instruction null check). |

## Returns

`void`

## Example

```ts
import { setLogger, defaultTextLogger } from "@snapcap/native";

setLogger(defaultTextLogger);   // built-in text formatter
setLogger((ev) => myJsonLogger.log(ev));   // custom handler
setLogger(undefined);   // disable
```
