Defined in: [transport/throttle.ts:107](https://github.com/CodingButter/snapcap-native/blob/e5c6ef042baa7d36bf620eee6b725ba6bbed1720/src/transport/throttle.ts#L107)

One throttle rule. Matches outbound URLs and gates them at a minimum
inter-call interval, with optional burst headroom.

## Remarks

First-match wins inside a [ThrottleConfig.rules](ThrottleConfig.mdx#rules) array — order
rules from most-specific to least-specific.
