# Class: FileDataStore

Defined in: [storage/data-store.ts:97](https://github.com/CodingButter/snapcap-native/blob/main/src/storage/data-store.ts#L97)

File-backed [DataStore](../interfaces/DataStore.mdx) implementation.

## Remarks

A single JSON file holds all entries; loaded into memory at construction;
eager flush on every `set` / `delete`.

Also exposes synchronous [FileDataStore.getSync](#getsync) /
[FileDataStore.setSync](#setsync) / [FileDataStore.keys](#keys) for the Web
Storage shims, which implement the synchronous Web Storage API.

## Example

```ts
import { SnapcapClient, FileDataStore } from "@snapcap/native";
const dataStore = new FileDataStore(".tmp/auth/auth.json");
const client = new SnapcapClient({ dataStore, username, password });
```
