Every public surface of `@snapcap/native`. Anything not listed here is internal and can change without notice.

For most apps you'll only need [`SnapcapClient`](/docs/api/snapcap-client), [`Conversation`](/docs/api/conversation), [`User`](/docs/api/user), and [`FileDataStore`](/docs/api/storage#filedatastore). The rest is escape hatches.

| Page | What's there |
|---|---|
| [SnapcapClient](/docs/api/snapcap-client) | The main class. Constructor, auth, friends, conversations, messaging, presence, low-level RPC. |
| [Conversation](/docs/api/conversation) | Domain object for a single chat. `sendText`, `sendImage`, `setTyping`, `markViewed`. |
| [User](/docs/api/user) | The `User` data class + factory methods. |
| [Messages & inbound stream](/docs/api/messages) | `DecodedMessage`, `Unsubscribe` — the types you touch from `client.onMessage`. |
| [Storage](/docs/api/storage) | `DataStore` interface, built-in stores, `StorageShim`, `CookieJarStore`, IDB helpers. |
| [Sandbox](/docs/api/sandbox) | `installShims`, `getSandbox`, the `Sandbox` class. Advanced bundle-interop only. |

## Imports at a glance

```ts
import {
  // Client + types
  SnapcapClient,
  type SnapcapClientOpts,

  // Domain objects
  Conversation,
  TypingActivity,
  ConversationViewState,
  type ConversationKind,
  type ConversationSummary,
  User,
  FriendSource,
  FriendAction,            // legacy enum; prefer FriendSource

  // Inbound message stream
  type DecodedMessage,
  type Unsubscribe,

  // Storage
  type DataStore,
  FileDataStore,
  MemoryDataStore,
  StorageShim,
  CookieJarStore,
  idbGet,
  idbPut,
  idbDelete,

  // Sandbox / bundle interop (advanced)
  installShims,
  getSandbox,
  isShimInstalled,
  Sandbox,
  type SandboxOpts,
  type InstallShimOpts,

  // Low-level helpers
  uuidToBytes,
  bytesToUuid,
  uuidToHighLow,
} from "@snapcap/native";
```

> The `FideliusIdentityBlob` type is also exported but **deprecated** — the SDK no longer mints or persists Fidelius identities (the chat bundle owns that end-to-end). Don't use it in new code. See [SnapcapClient → Types](/docs/api/snapcap-client#fideliusidentityblob-deprecated).
