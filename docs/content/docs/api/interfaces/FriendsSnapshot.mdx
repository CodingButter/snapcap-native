# Interface: FriendsSnapshot

Defined in: [api/friends/types.ts:235](https://github.com/CodingButter/snapcap-native/blob/main/src/api/friends/types.ts#L235)

A point-in-time view of the entire friend graph — mutuals + pending
requests in both directions.

## Remarks

Returned by [IFriendsManager.snapshot](IFriendsManager.mdx#snapshot) (canonical) and the
underlying source for [IFriendsManager.list](IFriendsManager.mdx#list),
[IFriendsManager.receivedRequests](IFriendsManager.mdx#receivedrequests), and
[IFriendsManager.sentRequests](IFriendsManager.mdx#sentrequests). The same object shape is
delivered to [IFriendsManager.onChange](IFriendsManager.mdx#onchange) subscribers whenever any
of the three slots mutates.
