# Interface: ReceivedRequest

Defined in: [api/friends.ts:224](https://github.com/CodingButter/snapcap-native/blob/main/src/api/friends.ts#L224)

An inbound friend request — someone has added the logged-in user and is
waiting for [IFriendsManager.acceptRequest](IFriendsManager.mdx#acceptrequest) or
[IFriendsManager.rejectRequest](IFriendsManager.mdx#rejectrequest).
