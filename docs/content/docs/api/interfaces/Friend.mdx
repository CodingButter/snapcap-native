# Interface: Friend

Defined in: [api/friends.ts:208](https://github.com/CodingButter/snapcap-native/blob/main/src/api/friends.ts#L208)

A friend in the logged-in user's social graph.

Superset of [User](FriendsUser.mdx) with friend-link metadata and (when surfaced
by the server) timestamps.
